Radius & elevation — almost none, by design
0px — in the TUI
terminal cells can't round. Blocks are flat tints; grouping comes from a 1-shade fill, never a border or shadow.
4px — web export max
the HTML export rounds blocks 4px. Treat 4px as the ceiling anywhere. No drop-shadows, no blur, no glow.
Elevation is conveyed by tint contrast against the canvas — not by Z-depth.
